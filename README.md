# test--name--shareapp-
Fast, secure, and resource-efficient file sharing via P2S2P architecture. test-name leverages chunked data streams and an optimized signaling layer to deliver seamless transfers across any network, regardless of hardware limitations.

## Local commands (from repository root)

Install app dependencies:

```bash
npm run install:app
```

Build:

```bash
npm run build
```

## Vercel

Install Vercel CLI:

```bash
npm run vercel:install
```

Run a Vercel build:

```bash
npm run vercel:build
```
